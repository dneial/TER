Ce projet à été réalisé avec la version 2021.3.18f1 de Unity.
Si des problèmes de compatibilité ont lieu avec une version ultérieure, veuillez tester le projet à l’aide de cette version.

INSTALLATION : 

1. Ouvrir un nouveau projet Unity

2. Extraire le contenu de l'archive CoralCraft.rar 

3. Glisser les dossiers "Coral" et "Terrain" dans le répertoire "Assets" Généré par défaut par votre nouveau projet Unity.


UTILISATION: 

1. Génération d'une unique structure Corallienne : 

	Algorithme L-système : 

	   -Dans la barre d’outils sélectionner le menu “Coral”
	   -Cliquer sur l’option “L-Système”
	   -Le menu de paramétrage de l’algorithme L-système s’ouvre alors

	Algorithme Space-Colonization : 

	   -Dans la barre d’outils sélectionner le menu “Coral”
	   -Cliquer sur l’option “Space-Colonization”
	   -Le menu de paramétrage de l’algorithme Space-Colonization s’ouvre alors

2.  Génération de multiples structures sur un terrain :

	-Dans la barre d’outils sélectionner le menu “Coral”
	-Cliquer sur l’option “Génération”
	-Un terrain sera généré automatiquement dans l’environnement.
	-Le menu de paramétrage de Génération multiple s’ouvre alors
	-L’option  “Randomgeneration” place les coraux de manière aléatoire sur le terrain.
	-L’option  “ProbaGeneration” prend en compte l’altitude du terrain pour placer les coraux.
